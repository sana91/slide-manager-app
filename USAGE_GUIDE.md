# 📖 Slide Manager 使用ガイド

## 🎯 概要

Slide Managerは、HTMLスライドを管理・登録・閲覧するためのWebアプリケーションです。
スライド（親）とページ（子）の階層構造でコンテンツを管理し、各ページはHTMLソースコードで直接編集できます。

## 🚀 クイックスタート

### 1. 新しいスライドを作成する

1. トップページまたはナビゲーションバーの「**新規作成**」をクリック
2. 必須項目を入力:
   - **スライド名**: わかりやすい名前（例: "2024年度プレゼンテーション"）
   - **スライドコード**: 英数字のみ、3〜50文字（例: "presentation2024"）
     - このコードがURLに使用されます: `/slide/presentation2024/`
     - 一意である必要があります（重複不可）
3. 任意項目を入力:
   - **説明**: スライドの概要
   - **タグ**: カンマ区切りで複数指定可能（例: "ビジネス, プレゼン"）
4. 「**スライドを作成**」をクリック

### 2. ページを追加する

1. スライド詳細ページで「**ページを追加**」をクリック
2. HTMLソースコードを入力:
   ```html
   <h1>タイトルページ</h1>
   <p>これは最初のスライドです。</p>
   ```
3. 「**ページを追加**」をクリック
4. ページ番号が自動で付与されます（1から順番に）

### 3. スライドを表示する

- スライド詳細ページの各ページカードの「**プレビュー**」をクリック
- または、直接URLにアクセス:
  - `/slide/{スライドコード}/{ページ番号}`
  - 例: `/slide/presentation2024/1`

### 4. プレゼンテーションモード

- スライド表示画面では:
  - **キーボードの ← / → キー**でページ移動
  - 画面上部にページ番号が表示されます
  - ナビゲーションバーで現在位置を確認できます

## 📋 機能詳細

### スライド管理

#### スライド一覧

- **検索**: スライド名、コード、説明で検索可能
- **表示情報**:
  - スライド名
  - スライドコード（URL）
  - ページ数
  - 最終更新日
  - タグ

#### スライド作成

- **必須項目**:
  - スライド名
  - スライドコード（英数字のみ、3〜50文字）
- **任意項目**:
  - 説明
  - タグ（カンマ区切り）

#### スライド編集

- スライド詳細ページの「**編集**」ボタンから
- 基本情報の変更が可能
- スライドコードの変更も可能（URLが変更されます）

#### スライド削除

- スライド詳細ページの「**削除**」ボタンから
- 確認ダイアログが表示されます
- 削除すると**すべてのページも削除**されます

### ページ管理

#### ページ追加

1. スライド詳細ページで「**ページを追加**」をクリック
2. HTMLソースコードを入力
3. ページ番号は自動で付与（最後のページ番号 + 1）

#### ページ編集

1. ページカードの「**編集**」ボタンをクリック
2. モーダルウィンドウでHTMLを編集
3. 「**更新**」をクリックして保存

#### ページ削除

1. ページカードの「**削除**」ボタンをクリック
2. 確認ダイアログで確認
3. 削除したページ番号は**再利用されません**

### スライド表示

#### ナビゲーション

- **キーボード**:
  - `←` (左矢印): 前のページへ
  - `→` (右矢印): 次のページへ
- **マウス**:
  - 画面上部の「前へ」「次へ」ボタン

#### 表示機能

- ページ番号表示（例: "3 / 10"）
- スライド名表示
- レスポンシブデザイン
- HTMLの自動サニタイゼーション（XSS対策）

## 💡 HTMLの書き方

### 基本的なHTML

```html
<h1>メインタイトル</h1>
<h2>サブタイトル</h2>
<p>本文のテキストです。</p>

<ul>
  <li>リスト項目1</li>
  <li>リスト項目2</li>
  <li>リスト項目3</li>
</ul>
```

### スタイル付きHTML

```html
<div style="text-align: center; padding: 2rem;">
  <h1 style="color: #0284c7; font-size: 3rem;">
    タイトル
  </h1>
  <p style="color: #64748b; font-size: 1.5rem;">
    サブタイトル
  </p>
</div>
```

### 画像の埋め込み

```html
<img 
  src="https://example.com/image.jpg" 
  alt="説明" 
  style="max-width: 100%; border-radius: 8px;"
/>
```

### レイアウト例

```html
<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 2rem;">
  <div>
    <h3>左側のコンテンツ</h3>
    <p>テキスト</p>
  </div>
  <div>
    <h3>右側のコンテンツ</h3>
    <p>テキスト</p>
  </div>
</div>
```

## 🎨 スタイリングのヒント

### デフォルトスタイル

スライド表示ページでは以下のデフォルトスタイルが適用されます:

- `<h1>`: 大きなタイトル（4xl）
- `<h2>`: 中サイズのタイトル（3xl）
- `<h3>`: 小サイズのタイトル（2xl）
- `<p>`: 本文テキスト（lg）
- `<ul>`, `<ol>`: リスト
- `<code>`: インラインコード
- `<pre>`: コードブロック

### カスタムスタイル

インラインスタイルで自由にカスタマイズできます:

```html
<div style="
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  padding: 3rem;
  border-radius: 1rem;
  text-align: center;
">
  <h1>グラデーション背景</h1>
</div>
```

## 📊 データ管理

### データの保存場所

- **LocalStorage**: ブラウザのローカルストレージに保存
- キー: `slides`
- データはブラウザごとに保存されます

### データのバックアップ

現時点ではエクスポート機能はありませんが、以下の方法でバックアップできます:

1. ブラウザの開発者ツールを開く（F12）
2. Consoleタブで実行:
   ```javascript
   console.log(localStorage.getItem('slides'))
   ```
3. 出力されたJSONをコピーして保存

### データのインポート

1. ブラウザの開発者ツールを開く（F12）
2. Consoleタブで実行:
   ```javascript
   localStorage.setItem('slides', 'バックアップしたJSON')
   ```
3. ページをリロード

## 🐛 よくある問題と解決方法

### スライドが表示されない

- **原因**: データがLocalStorageに保存されていない
- **解決**: ブラウザのLocalStorageが有効か確認

### スライドコードが重複しているエラー

- **原因**: 同じスライドコードが既に存在
- **解決**: 別のユニークなコードを使用

### ページ番号がずれる

- **原因**: ページを削除した場合、番号は再利用されません
- **仕様**: これは意図的な動作です

### HTMLが正しく表示されない

- **原因**: 危険なタグがサニタイズされた
- **解決**: 安全なHTMLタグのみ使用してください
  - 許可: `<h1>`, `<p>`, `<div>`, `<img>`, `<ul>`, etc.
  - 禁止: `<script>`, `<iframe>`, etc.
